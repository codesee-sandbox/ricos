## Usage

```jsx
import { RicosEditor } from 'ricos-editor';
import { pluginGiphy } from 'wix-rich-content-plugin-giphy';
import 'ricos-editor/dist/styles.min.css';
import 'wix-rich-content-plugin-giphy/dist/styles.min.css';

/* Editor */
<RicosEditor plugins={[pluginGiphy(GiphyConfig)]} />

/* Viewer */
<RicosViewer plugins={[pluginGiphy(GiphyConfig)]} />
```

### Giphy Config API

| key              | Required?                         | Type   | Default                                   | Description                                                                                                   |
| ---------------- | --------------------------------- | ------ | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `giphySdkApiKey` | V                                 | String | X                                         | <a href="https://support.giphy.com/hc/en-us/articles/360020283431-Request-A-GIPHY-API-Key"> GIPHY API key</a> |
| `sizes`          | { desktop?: size, mobile?: size } |        | {desktop: 'original', mobile: 'original'} | Control GIF size - (original / downSizedSmall)                                                                |

<br />
